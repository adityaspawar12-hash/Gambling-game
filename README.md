# Simulator

A terminal-based Python implementation of the classic Matka game. This project simulates the full draw process, including Open/Close Pannas, Ank calculation, and Jodi formation.

## About the Game

Satta Matka is a numbers game that originated in the 1950s. This simulator recreates the "OG" experience:
- Panna: 3 random digits displayed in ascending order.
- Ank: The last digit of the sum of the Panna.
- Jodi: The combination of the Open Ank and Close Ank.

## Features

- Full Draw Format: Displays results in the professional `123-67-458` format.
- Automatic Sorting: Numbers are sorted automatically to match authentic Matka charts.
- Multiple Bet Types: Supports Single (9x), Jodi (90x), and Panna (280x) betting logic.
- Educational Insights: Includes "Winning Suggestions" if you lose to show the correct combination.
